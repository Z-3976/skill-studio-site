# xiaohongshu-biji

## 使用场景

适用于以下任务：

- 小红书笔记
- 小红书团购转化笔记
- 小红书探店种草笔记
- 小红书轻IP日常笔记
- 小红书干货清单笔记
- 小红书笔记封面图
- 根据门店信息、产品信息、主题色做小红书笔记图

不适用于以下任务：

- 普通聊天
- 代码开发
- 直播话术
- 抖音短视频脚本
- 非小红书视觉设计任务

## 输入信息要求

优先收集以下信息：

- 门店名 / 位置
- 产品或主题
- 门店卖点 / 产品权益
- 目标人群
- 笔记目的
- 参考风格或参考链接
- 可选主题色
- 可选 Logo
- 可选参考图

若用户明确指定输出为“小红书笔记图”，则优先补齐：

- 主题色
- 参考风格
- Logo
- 参考图

## 执行流程

### 第一步：检查输入是否完整

先判断用户当前要的是：

- 小红书笔记
- 小红书笔记图

再检查是否已提供：

- 门店名 / 位置
- 产品或主题
- 门店卖点 / 权益
- 目标人群
- 笔记目的
- 参考风格
- 主题色
- Logo
- 参考图

其中 Logo、主题色、参考图可根据任务类型作为可选项，但若用户明确要封面图，优先补齐。

### 第二步：如果缺信息，先用最少问题补齐

只追问当前结果最关键的信息，尽量一次性问完。

示例：

`请补这几项：1. 门店名或位置 2. 产品或主题 3. 核心卖点/权益 4. 想走哪种路线（团购转化 / 探店种草 / 轻IP日常 / 干货清单） 5. 如果要封面图，再补主题色或参考图。`

### 第三步：整理成统一参数

整理出以下统一内容：

- 输出类型：小红书笔记 / 小红书笔记图
- 路线：团购转化 / 探店种草 / 轻IP日常 / 干货清单
- 门店定位
- 产品主题
- 目标人群
- 内容目标
- 核心卖点 / 权益
- 风格参考
- 主题色约束
- Logo 和参考图使用规则

### 第四步：如果环境支持图片生成，直接生成结果

若用户要的是“小红书笔记图”，且当前环境支持图片生成，则直接生成封面图。

默认规则：

- 3:4 竖版
- 主标题放中上区域
- 避开顶部和底部 UI 遮挡区
- 字少、字大、对比强
- 若用户给主题色，保持大体色差不变
- 若给 Logo，只做弱品牌露出
- 团购转化型可突出价格数字
- 轻IP / 探店种草型不要强促销海报感

### 第五步：如果环境不支持图片生成，则输出最终中文生图提示词

若用户要的是“小红书笔记图”，但当前环境不支持直接生图：

- 只输出“最终生图提示词”
- 不输出冗长解释
- 不输出多版方案

若用户要的是“小红书笔记”，则直接输出成稿，不额外讲流程。

## 输出规则

- 若是“小红书笔记”：
  - 直接输出成稿
  - 默认结构为：
    - 笔记路线
    - 标题 3 版
    - 开头钩子
    - 正文成稿
    - 分段小标题
    - 结尾互动句
    - 标签建议
    - 封面文案

- 若是“小红书笔记图”且可直接生图：
  - 直接生成图片
  - 简短说明已按 `3:4 + 标题避开UI遮挡区` 执行

- 若是“小红书笔记图”且不可直接生图：
  - 只输出 `最终生图提示词`

- 不要先长篇分析再开始生成

## 设计规范

- 风格优先参考用户给的样板图
- 默认尺寸 3:4 竖版
- 主标题落在中上区域
- 避开小红书顶部和底部 UI 遮挡区
- 字少、字大、对比强
- 用户主题色作为主色
- 若用户给定主题色，整体颜色大体色差不要明显变化
- Logo 只做弱品牌露出，不强行做大 Logo 海报
- 团购转化型重点突出价格、权益、适合人群
- 轻IP / 种草型重点突出体验感、氛围感、真实感
- 干货清单型更像建议页、清单页、避坑页
- 整体要清晰、干净、有点击欲，但不要像直播预告板

## 示例

### 示例 1：团购转化笔记

用户输入：

`门店在南京新街口，产品是99元健身月卡，卖点有免费淋浴、免费停车、巡场教练、体测，目标人群是附近上班族，帮我写一篇团购转化型小红书笔记。`

期望行为：

- 判断为“小红书笔记”
- 路线为“团购转化笔记”
- 允许价格、权益、适合人群、到店动作
- 输出标题 3 版 + 正文成稿 + 封面文案

### 示例 2：探店种草封面图

用户输入：

`给我做一个小红书笔记图，主题是新手减脂体验，主题色薄荷青，风格清爽一点，Logo 可弱露出。`

期望行为：

- 判断为“小红书笔记图”
- 默认按 3:4 竖版生成
- 保持薄荷青大体色差不变
- 不做强促销海报感

### 示例 3：轻IP日常笔记

用户输入：

`写一篇教练口吻的小红书日常笔记，内容是最近很多新手第一次来店里会紧张。`

期望行为：

- 判断为“轻IP日常笔记”
- 不写价格、不写库存、不写逼单
- 更像真实门店口吻和体验分享

### 示例 4：环境不支持生图时

用户输入：

`帮我做一张小红书封面图，主题是99元健身月卡，主题色蓝色。`

期望行为：

- 若当前环境不支持直接生图
- 只输出：
  - `最终生图提示词：...`
