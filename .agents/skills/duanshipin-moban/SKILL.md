# duanshipin-moban

## 使用场景

适用于以下任务：

- 根据本地文件生成短视频文案
- 根据飞书文档、表格、培训课件整理短视频模版
- 根据参考视频风格改写成新脚本
- 为健身房、本地生活、门店账号生成抖音脚本
- 生成口播稿、分镜、标题、封面文案、字幕文案、发布建议
- 生成多场景拍摄脚本
- 根据不同账号路线输出不同风格的脚本

本 Skill 默认覆盖以下四类：

- 营销短视频
- 官方轻IP（3km大众熟人）
- 素人氛围号
- 预热视频 / 核销视频 / 引流视频

四类路线必须严格区分，不要混写：

- 用户指定 `营销视频` 时，按纯营销路线写
- 用户指定 `轻IP` 或 `3km大众熟人` 时，禁止混入营销视频写法
- 用户指定 `素人氛围号` 时，禁止写成硬广
- 用户指定 `预热 / 核销 / 引流` 时，优先解决行动转化问题

不适用于以下任务：

- 普通聊天
- 代码开发
- 非短视频任务
- 纯平面设计任务

## 输入信息要求

优先收集以下信息：

- 本地文件：产品介绍、活动方案、门店资料、课程资料、培训课件、用户评价、历史脚本、表格、文档
- 外部链接：飞书文档、网页、资料页、参考视频链接
- 内容目标：引流、转化、建立信任、种草、活动通知、核销、预热、品牌曝光、门店宣传
- 账号类型或视频路线：营销 / 官方轻IP / 素人氛围 / 预热核销引流
- 行业和对象：健身房、瑜伽馆、门店、本地生活、课程、套餐、月卡、私教
- 可选补充：地区、门店名、价格、权益、目标人群、发布时间、语气偏好

若用户没有明确说明路线，则根据素材内容和目标自动判断最匹配的一类。

## 执行流程

### 第一步：先读取用户提供的文件和链接

必须先读取用户提供的材料，再开始写文案。

优先读取：

- 当前项目中的本地文件
- 用户明确提到的文档
- 培训课件
- 飞书文档和表格
- 文档中挂载的参考视频或作品链接
- 用户指定的参考账号和代表作品

如果链接需要登录、验证码或额外交互：

- 先进入对应页面
- 需要用户登录时，请用户完成登录
- 需要用户处理验证码时，暂停并等待用户操作

不要在没有读取素材的前提下直接凭空写稿。

### 第二步：提炼素材中的关键信息

从已读取材料中提炼：

- 产品或服务名称
- 价格和权益
- 场馆卖点
- 用户痛点
- 适合人群
- 真实证据
- 品牌口吻
- 门店调性
- 可用于视频的具体场景
- 所处运营阶段：预热 / 上架 / 引流 / 核销 / 复购 / 活动通知

优先整理以下内容来源：

- 用户购买理由
- 用户体验反馈
- 门店相关介绍
- 行业知识
- 关键人群热点话题
- 门店经营理念

### 第三步：判断应走哪一种模板路线

根据用户目标和素材内容，在以下四类中选择最合适的一类；若有必要，可同时给出多类版本。

#### 1. 营销短视频

适合：

- 强引流
- 强点击
- 强转化
- 团购、套餐、活动、促销、痛点型内容
- A3 / A4 人群种草到下单

这是纯营销路线。

若用户明确说要 `营销视频`，则默认允许：

- 出现价格
- 出现原价 / 现价对比
- 出现价格拆分
- 出现权益包拆解
- 出现下单引导
- 出现库存、名额、活动时效提醒

不要把轻IP、信任感长铺垫、日常聊天感当成营销视频主体。

根据培训课件的结构，优先使用以下骨架：

- 门店位置
- 行业
- 门店优势
- 用户痛点 / 行业对比
- 团单内容拆解
- 左下角或结尾引导下单

开头必须有：

- 噱头
- 痛点钩子
- 反差
- 价格利益点

默认优先使用以下三种价格表达方式：

1. 原价和现价对比
2. 把总价拆成日均价格
3. 把价格和权益包直接绑定，强调性价比

#### 2. 官方轻IP（3km大众熟人）

适合：

- 门店官方号
- 品牌号
- 蓝V账号
- 老板号 / 店长号 / 员工号
- 需要建立信任感、熟悉感、持续更新感的内容

这是非营销路线。

若用户明确说要 `轻IP`、`官方轻IP`、`3km大众熟人`，则默认禁止：

- 出现价格
- 出现原价 / 现价
- 出现团购促单
- 出现库存不多了
- 出现名额不多了
- 出现活动不是随时都有
- 出现点击左下角下单
- 出现立刻囤一下
- 出现任何硬促销指令

根据培训课件和已读取账号，核心不是硬卖货，而是：

- 3km 大众熟人
- 软营销
- 泛内容 + 泛垂直
- 真人口述
- 真诚表达
- 门店长期存在感

参考风格基线：

- `CC GYM官方号`
- `CCGYM（光大店）官方号`

已验证的轻IP共性：

- 口吻口语化，不生硬念稿
- 强调门店服务、器械、环境、活动、规则、福利和日常更新
- 以“我们”“大家”为核心，像店长、主理人、运营、教练在说话
- 强调真诚、透明、说到做到、有求必应、重视会员体验
- 常见内容结构包括：
  - 新器械更新
  - 一镜到底看场馆
  - Q&A
  - 活动通知
  - 核销提醒
  - 营业时间说明
  - 门店细节优化
  - 会员权益和服务说明
  - 场馆优缺点坦诚表达

#### 3. 素人氛围号

适合：

- 情绪感内容
- 真实体验分享
- 生活方式种草
- 场景氛围型表达

根据培训课件，内容核心是：

- 场景
- 情绪
- 音乐
- 细节
- 轻文案
- 共鸣
- 流行感
- 治愈 / 解压 / 自律 / 松弛

内容特征：

- 更像真实人分享，不像官方宣传
- 更强调场景、情绪、生活细节、音乐感、松弛感
- 轻文案、轻卖货、强氛围

#### 4. 预热视频 / 核销视频 / 引流视频

适合：

- 活动预热
- 上架前蓄水
- 团购核销
- 引导用户来店
- 引导用户先核销再开卡
- 未核销用户召回

根据培训课件，这一类内容要非常直接，核心是：

- 提醒用户有效期
- 说明核销收益
- 给出核销理由
- 告诉用户怎么做
- 明确行动路径

优先使用以下骨架：

- 这件事和谁有关
- 不做会损失什么
- 现在来能得到什么
- 核销或到店步骤
- 明确截止时间或提醒动作

### 第四步：整理成统一脚本参数

在开始写之前，整理为统一参数：

- 账号路线：营销 / 官方轻IP / 素人氛围 / 预热核销引流
- 视频目标：引流 / 转化 / 信任 / 种草 / 通知 / 招募 / 预热 / 核销
- 视频形式：真人口播 / 视频+配音 / 图片+配音 / Vlog / 一镜到底 / 探店 / Q&A / 对话 / 清单 / 设备讲解 / 场馆展示
- 拍摄结构：单场景 / 多场景
- 语气：直接、真诚、轻松、专业、带一点人味
- 核心钩子
- 核心卖点
- 核心场景
- 结尾动作

### 第五步：输出短视频文案

默认输出应尽量接近可直接开拍的状态。

优先输出以下内容：

- 视频路线
- 视频标题
- 前3秒钩子
- 完整口播稿
- 分镜建议
- 多场景拍摄安排
- 屏幕字幕建议
- 封面文案
- 结尾引导语
- 标签方向

若用户只要其中某一项，则按用户要求缩减。

## 输出规则

- 默认先基于素材写，不凭空发散
- 不要先长篇分析，再给一句文案
- 输出要偏成片，不要像写作文
- 若素材明显适合蓝V轻IP，优先使用 `官方轻IP（3km大众熟人）`
- 若素材明显适合强种草，优先使用 `素人氛围号`
- 若素材明显适合转化或活动促销，优先使用 `营销短视频`
- 若素材明显处于活动节点、上架节点、核销节点，优先使用 `预热视频 / 核销视频 / 引流视频`
- 若用户没指定数量，默认输出 `1条主稿 + 2条备选方向`
- 若用户要效率优先，可只输出 `1条最优版本`
- 若用户明确指定路线，必须严格遵守该路线，不要混入其他路线的写法
- `营销短视频` 和 `官方轻IP（3km大众熟人）` 必须强隔离，不能混写
- `轻IP`、`3km大众熟人`、`素人氛围号` 默认不出现价格
- 若用户没有特别说明，优先输出可直接拍摄的多场景版本

## 真人口述规范

根据培训课件，真人口述优先遵守以下要求：

- 真诚
- 有情绪起伏
- 有原生态感
- 随意好于刻意
- 活人感强
- 不要照稿死念

表达上优先像真实人在说话，而不是品牌在念通知。

## 多场景拍摄规范

默认优先按多场景去组织脚本，不要只给一整段口播文字。

多场景输出至少应包含：

- 场景编号
- 场景位置
- 画面内容
- 对应口播
- 建议镜头

优先使用以下常见场景：

- 门头 / 门店外景
- 前台 / 进门
- 器械区全景
- 具体器械特写
- 教练 / 员工互动
- 会员训练状态
- 休息区 / 喝水 / 擦汗 / 收拾东西
- 淋浴区 / 更衣区 / 停车 / 走廊等细节区域
- 结尾中景或边走边说收尾

多场景拍摄要求：

- 画面不要一直站桩式口播
- 能边走边说，就不要一直正对镜头念稿
- 同一条视频里尽量有远景、中景、近景切换
- 口播和画面要互相解释，不要完全脱节
- 门店类内容优先拍真实空间、真实器械、真实动线、真实互动
- 轻IP类内容优先拍自然交流感，不要像广告棚拍

### 多场景默认结构

- 场景1：开头钩子场景
- 场景2：问题或痛点场景
- 场景3：卖点或细节场景
- 场景4：补充证明场景
- 场景5：结尾互动场景

### 不同路线的多场景偏好

营销短视频优先场景：

- 门头 / 场馆全景
- 器械区
- 权益对应画面
- 团单价值对应画面
- 结尾引导下单画面

官方轻IP优先场景：

- 门店日常
- 老板 / 店长 / 员工自然出镜
- 场馆真实细节
- 通知、更新、答疑相关画面
- 会员和教练真实互动

素人氛围号优先场景：

- 光线舒服的局部空间
- 训练中的小细节
- 呼吸、擦汗、喝水、走路、拉伸
- 情绪氛围和生活感镜头

预热视频 / 核销视频 / 引流视频优先场景：

- 能说明活动节点的画面
- 到店动线
- 核销步骤提示
- 门店权益或福利对应画面
- 收尾行动提示画面

## 营销短视频写法规范

- 开头先给钩子
- 再给用户痛点或价格利益点
- 中段快速证明门店优势
- 明确团单或产品内容
- 结尾留动作，推进下单或咨询

### 营销短视频允许出现的价格表达

- 原价 `259`，现在 `99`
- `99` 除以 `30` 天，日均只要 `3` 块多
- 一顿火锅 / 一杯饮品 / 一次随手消费，不如来场馆练一个月
- `99` 包含哪些权益，直接拆开说清楚

### 营销短视频常用性价比表达

- 原价对比型
- 日均价格型
- 权益打包型
- “少花钱先开始”型

### 营销短视频结尾动作规范

若用户要求营销视频，默认可以使用强引导动作：

- 左下角点击位置下单
- 看看链接还有没有
- 没有的话赶紧囤一下
- 库存不多了
- 活动不是随时都有
- 现在不下，后面未必还有这个价格
- 想要的直接点左下角

常见适配结构：

- 你以为 / 实际上
- 别再 / 赶紧
- 如果你是某类人
- 这件事我们必须说清楚
- 一条视频讲明白
- 这个价格到底值不值

## 官方轻IP（3km大众熟人）写法规范

以 `CC GYM官方号` 和 `CCGYM（光大店）官方号` 的已验证风格为基准：

- 不装腔，不端着，不官话套话
- 像主理人、店长、运营、教练自然对用户说话
- 真实表达门店变化、设备更新、运营细节、服务承诺、活动提醒
- 不强行搞大情绪，重点是可信、持续、有温度
- 标题常带具体对象或具体问题，例如：
  - 如果第一次来店里该怎么办
  - 大家最关心的问题
  - 我们又更新了什么
  - 核销提醒
  - 一镜到底看哪里
- 内容里要有“大家”“会员”“我们”“来店里”“欢迎来体验”这类真实门店交流语言
- 可以带轻微幽默感，但不要油腻
- 允许讲缺点、讲细节、讲规则、讲改进，这会增强信任感
- 允许做长一点的视频，但节奏仍要清楚

### 官方轻IP禁区

这一类内容默认不允许出现：

- 任何价格
- 原价和现价对比
- 团单拆解
- 名额稀缺
- 库存不多
- 活动快结束
- 点击左下角下单
- 看看链接还有没有
- 赶紧囤
- 强促单和强逼单

### 官方轻IP正确方向

这一类更适合写：

- 大家最近最关心的问题
- 如果第一次来门店该怎么办
- 我们最近又更新了什么
- 门店哪里又优化了
- 为什么我们这样做
- 一些对会员有帮助的提醒
- 一些真实的门店日常和真实想法

### 3km大众熟人优先内容池

- 用户购买理由
- 用户体验反馈
- 门店位置和区域
- 门店重要通知
- 门店装修和更新进度
- 门店经营理念
- 行业知识和避坑
- 老会员真实聊天
- 员工和老板日常

### 老板号 / 员工号建议

老板号优先拍：

- 个人巡店日常
- 跟教练 / 员工互动
- 创业 / 开店心得
- 跟老会员真实聊天
- 自己训练日常
- 门店小问题和解决过程
- 门店小福利

员工号优先拍：

- 自己训练日常
- 带会员上课状态
- 健身小干货
- 会员变化记录
- 减脂 / 饮食小分享
- 门店日常

## 素人氛围号写法规范

- 开头像真实感受
- 不像品牌公告
- 轻文案、轻推销、重氛围
- 强场景感、情绪感、生活感
- 多用细节来建立代入感
- 优先写“我今天看到 / 我刚练完 / 我突然理解了 / 原来这里最舒服的是”
- 默认不出现价格和硬促销

## 预热视频 / 核销视频 / 引流视频写法规范

### 预热视频

- 用来提前蓄水
- 说明什么时候上、为什么值得等、谁最适合
- 可以先释放一部分价值点，但不要一次说完

### 核销视频

- 要直接
- 要明确有效期
- 要说明先核销的价值
- 要告诉用户怎么做
- 内容必须清楚，不要太绕

优先表达：

- 你的团购快到期了
- 先来核销，后续再开卡也来得及
- 现在来能额外拿到什么

### 引流视频

- 要抓痛点
- 要讲地区和门店优势
- 要讲产品内容
- 要留下明确动作，例如评论、私信、点击、小黄车、到店

## 路线冲突处理规则

当用户同时给出多个目标时，按以下优先级判断：

- 如果用户明确说 `营销视频`，就写纯营销，不要混入轻IP口吻
- 如果用户明确说 `轻IP` 或 `3km大众熟人`，就写信任感和日常感，不要出现价格和促单
- 如果用户明确说 `素人氛围号`，就写场景和情绪，不要变成蓝V通知口吻
- 如果用户明确说 `核销视频`，就优先写有效期、提醒、行动路径和核销收益

## 发布与结构补充规范

根据培训课件，默认补充以下执行偏好：

- 视频封面尽量统一、整洁、干净
- 每个视频尽量带上行业和内容相关话题
- 保持稳定更新频率，不要过密也不要断更太久
- 默认建议每日至少 1 条
- 开头优先抢 2 秒和 5 秒停留
- 所有脚本都要优先考虑完播、互动、点击和转化

## 示例

### 示例 1：根据门店资料生成官方轻IP

用户输入：

`根据我发的门店介绍和器械更新表，帮我写一条官方轻IP口播，主题是新器械到了。`

期望行为：

- 先读取门店介绍和器械更新表
- 提炼出门店卖点和设备亮点
- 按 `官方轻IP（3km大众熟人）` 输出
- 风格自然、口语化、可信，不要像广告腔

### 示例 2：根据活动资料生成营销短视频

用户输入：

`根据这个月卡活动方案，给我写3条营销短视频文案，目标是团购转化。`

期望行为：

- 先读取活动方案
- 识别为 `营销短视频`
- 输出 3 条不同钩子方向的转化稿
- 允许出现价格、价格拆分、权益打包和左下角下单引导

### 示例 3：根据体验内容生成素人氛围号

用户输入：

`根据我发的体验记录，写一条像真实分享的种草口播。`

期望行为：

- 先读体验记录
- 识别为 `素人氛围号`
- 输出更像生活分享的脚本

### 示例 4：生成核销视频

用户输入：

`给未核销的月卡用户写一条核销提醒短视频。`

期望行为：

- 先明确产品有效期和核销利益点
- 识别为 `预热视频 / 核销视频 / 引流视频`
- 输出直接、清楚、带行动路径的脚本
- 同时输出适合核销提醒的多场景安排

### 示例 5：必须先读链接和视频

用户输入：

`参考我发的飞书文档和里面的视频，帮我写一套短视频模版。`

期望行为：

- 先打开飞书文档
- 读取文档中的文字、表格和参考视频链接
- 提炼共性后再输出模版
- 不要跳过“先读取素材”这一步

### 示例 6：轻IP禁止营销属性

用户输入：

`给我写一条轻IP口播，主题是99元月卡。`

期望行为：

- 可以围绕“为什么最近很多人来问月卡”“为什么建议先开始锻炼”去写
- 但不要直接写价格、原价现价、库存、下单、囤券
- 内容重点放在真实表达、门店日常、用户感受和信任感

### 示例 7：营销视频允许价格拆分和下单引导

用户输入：

`给我写一条99元健身月卡营销视频。`

期望行为：

- 允许写 `原价259，现在99`
- 允许写 `99除以30天，日均3块多`
- 允许写 `99包哪些权益`
- 允许写 `左下角点链接下单`
- 允许写 `库存不多了`、`活动不是随时都有`

### 示例 8：轻IP默认输出多场景脚本

用户输入：

`给我写一条轻IP口播，主题是第一次来健身房别太紧张。`

期望行为：

- 输出轻IP路线文案
- 不出现价格和促销
- 同时拆出多场景拍摄安排
- 至少给出门口、器械区、互动区、结尾区几个场景
